Explicación por bloque Encabezado del Documento HTML Ejercicio Práctico preparatorio Examen final Módulo Formativo 1 explicación : Declara que el documento es HTML5. : Define el idioma del contenido del documento como español. : Establece la codificación de caracteres del documento como UTF-8. : Hace que la página sea responsive ajustando el ancho al dispositivo del usuario. : Define el título de la página que aparecerá en la pestaña del navegador. <link rel="stylesheet" href="css/style.css">: Enlaza el archivo CSS externo para los estilos de la página. Cuerpo del Documento HTML <body> <header class="header"> <h1>Mi Blog</h1> <nav class="nav-contenedor"> <a class="mi-blog" href="index.html">Mi Blog</a> <a class="noticias" href="noticias.html">Noticias</a> <a class="login" href="login.html">Login</a> <a class="registro" href="registro.html">Registro</a> </nav> </header> explicación <body>: Inicia el cuerpo del documento. <header class="header">: Define el encabezado de la página. <h1>Mi Blog</h1>: Título del blog. <nav class="nav-contenedor">: Contenedor de navegación con enlaces a diferentes secciones de la página. Enlaces <a>: Enlaces de navegación a distintas páginas (index.html, noticias.html, login.html, registro.html). Contenedor SVG y Contenido Principal <div class="svg"> <svg id="mi-svg" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-linecap="round" stroke-linejoin="round" width="24" height="24" stroke-width="2"> <path d="M12 5l0 14"></path> <path d="M5 12l14 0"></path> </svg> </div> <main id="main-content"> <!-- Aquí se insertarán los artículos --> </main> explicación <div class="svg">: Contenedor del SVG. <svg id="mi-svg" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-linecap="round" stroke-linejoin="round" width="24" height="24" stroke-width="2">: Define un gráfico SVG con las propiedades especificadas. <path d="M12 5l0 14"></path> y <path d="M5 12l14 0"></path>: Definen las líneas vertical y horizontal del SVG. <main id="main-content">: Contenedor principal donde se insertarán los artículos dinámicamente. Modal y Formulario <dialog id="mi-modal"> <form method="dialog" class="modal-form"> <label for="titulo">Título:</label> <input type="text" id="titulo" name="titulo"> <label for="texto">Texto:</label> <textarea id="texto" name="texto" rows="10" cols="30"></textarea> <div class="modal-buttons"> <button id="agregar" type="button">Agregar</button> <button id="cerrar" type="button">Cerrar Modal</button> </div> </form> </dialog> explicación <dialog id="mi-modal">: Define un cuadro de diálogo que sirve como modal para agregar artículos. <form method="dialog" class="modal-form">: Formulario dentro del modal que se usa para agregar artículos. Etiquetas <label> y campos de entrada <input>/<textarea>: Campos para ingresar el título y texto del artículo. <div class="modal-buttons">: Contenedor para los botones del formulario. Botones <button>: Botones para agregar el artículo y cerrar el modal. Enlace a JavaScript <script src="js/index.js"></script> explicación <script src="js/index.js">: Enlaza el archivo JavaScript externo que contiene las funcionalidades interactivas del sitio.
